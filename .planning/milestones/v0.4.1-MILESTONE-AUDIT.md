## Tech Debt

### Phase 02: Validation and Safety

- **`formatResult()` does not render `warning` field to MCP caller text** — The `CrossReviewResult.warning` field is correctly set in the engine when content exceeds ~50K tokens (SAFE-01), but `formatResult()` in `src/index.ts` never reads it. Users submitting large content will not see the warning in their MCP client's text response. The data exists on the result object but is dropped at the display layer. Fix: add `if (result.warning) { lines.push("WARNING: " + result.warning) }` to `formatResult()`.
